CSS Resets

CSS reset components <CssReset*> provide a handy way to include CSS reset in your app. You simply need to render the css reset component somewhere in your JSX tree for it to take effect.

import CssResetEricMeyer from 'libreact/lib/reset/CssResetEricMeyer';

const App = () =>
  <div>
    <CssResetEricMeyer />
    { /* ... */ }
  </div>

When you un-mount a CSS reset component it will also remove the CSS automatically.

Below is a list of included CSS resets.

  • <CssResetEricMeyer>
  • <CssResetEricMeyerCondensed>
  • <CssResetMinimalistic>
  • <CssResetMinimalistic2>
  • <CssResetMinimalistic3>
  • <CssResetPoorMan>
  • <CssResetShaunInman>
  • <CssResetSiolon>
  • <CssResetTantek>
  • <CssResetUniversal>
  • <CssResetYahoo>
